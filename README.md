# libnginx-mod-http-fastdfs
FastDFS nginx module of dynamic.

## How to?
```
tar -zvxf libnginx-mod-http-fastdfs-6.12.3.tar.gz
sudo ./nginx-add-ngx_http_fastdfs_module.patch.sh
```
